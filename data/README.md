# Данные

Файл датасета **не включён в репозиторий** — данные являются конфиденциальными
материалами курса НИЯУ МИФИ (Skillfactory).

## Как получить данные

Файл предоставляется в рамках курса «Классическое машинное обучение» НИЯУ МИФИ (Skillfactory).
Название файла: `Данные_для_курсовои_Классическое_МО.xlsx`

## Куда положить файл

Положите файл в эту папку (`data/`):

```
data/
└── Данные_для_курсовои_Классическое_МО.xlsx
```

Все ноутбуки ищут датасет автоматически — сначала в текущей папке,
потом в папке `data/`, потом в домашней директории.

## Описание датасета

| Параметр | Значение |
|----------|----------|
| Строк (молекул) | 1 001 |
| Столбцов | 214 |
| Признаков (дескрипторов) | 210 |
| Пропущенных значений | 36 (в 3 строках) |

### Целевые переменные

| Переменная | Тип | Описание |
|------------|-----|----------|
| `IC50, mM` | float | Минимальная ингибирующая концентрация против вируса гриппа |
| `CC50, mM` | float | Цитотоксическая концентрация для 50% клеток |
| `SI`       | float | Индекс селективности: SI = CC50 / IC50 |

### Признаки

Все 210 признаков — числовые молекулярные дескрипторы, вычисленные с помощью
библиотеки RDKit. Они описывают физико-химические свойства молекул:
топологические индексы, счётчики фрагментов, электростатические характеристики,
дескрипторы растворимости и т.д.
